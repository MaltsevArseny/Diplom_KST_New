## 🚀 Запуск приложения

Дважды кликните по файлу **`run.bat`** в корневой папке проекта. Скрипт проверит наличие Java 21 или выше, соберёт проект и запустит приложение.

Для быстрого запуска без пересборки используйте **`dist/DigitalHub.bat`**.

---

## 🔐 Вход в панель администратора

Запустите приложение и войдите с учётной записью, которая имеет роль **Администратор**. После входа автоматически откроется панель администратора — она отличается от обычного интерфейса покупателя.

Если вам нужно создать нового администратора, это можно сделать из раздела «Пользователи» (см. ниже).

## 🧭 Интерфейс администратора

### Верхняя панель

Наверху расположены: логотип приложения, имя текущего пользователя, бейдж **ADMIN** и кнопки управления:

| Элемент | Клавиша | Что делает |
|---|---|---|
| 🚪 Выйти | `Alt+Q` | Выход из аккаунта |
| ❓ Справка | `F1` | Открывает это руководство |

Окно можно перемещать, перетаскивая за верхнюю панель. Двойной клик по ней — развернуть на весь экран.

### Боковое меню

Слева расположены четыре основных раздела:

| Раздел | Клавиша | Что внутри |
|---|---|---|
| 🏷 Товары | `Alt+1` | Добавление, редактирование и удаление товаров |
| 📋 Заказы | `Alt+2` | Просмотр и обработка заказов покупателей |
| 👥 Пользователи | `Alt+3` | Управление аккаунтами, блокировка, создание новых админов |

## 🏷 Управление товарами

В этом разделе находится таблица всех товаров магазина.

### Поиск

Вверху есть строка поиска — вводите название или описание, и таблица отфильтруется автоматически. Также можно выбрать категорию из выпадающего списка. Кнопка **«Сбросить»** возвращает все фильтры в исходное состояние.

### Как добавить новый товар

1. Нажмите кнопку **«＋ Добавить товар»**
2. Заполните поля:

| Поле | Что вводить | Нужно заполнить? |
|---|---|---|
| Название | Название товара (должно быть уникальным) | Да |
| Категория | Выберите из списка | Да |
| Цена (₽) | Стоимость (число больше нуля) | Да |
| Остаток | Сколько штук на складе | Да |
| Описание | Краткое описание товара | Нет |
| Характеристики | Технические параметры | Нет |

3. Нажмите **«✅ Сохранить»**

### Как отредактировать товар

Нажмите **«✏ Редактировать»** рядом с нужным товаром, измените данные и сохраните.

### Как удалить товар

Нажмите **«🗑 Удалить»** и подтвердите действие. Имейте в виду: если товар входит в чей-то активный заказ, лучше просто обнулить остаток, а не удалять.

### Какие есть категории

Процессоры · Видеокарты · Материнские платы · Оперативная память · Накопители · Блоки питания · Корпуса · Охлаждение · Периферия · Мониторы · Сетевое оборудование

## 📋 Управление заказами

Здесь отображаются все заказы покупателей.

### Таблица заказов

В таблице видно: номер заказа, дату, имя покупателя, текущий статус и сумму. Можно отфильтровать заказы по статусу или найти по номеру/имени покупателя.

### Как посмотреть подробности

Кликните по строке заказа — откроется окно с полной информацией:
- Адрес доставки, телефон, комментарий покупателя
- Список товаров (с количеством и ценой)
- Текущий статус и вся история его изменений

### Как изменить статус заказа

1. Откройте заказ (кликните по строке)
2. Выберите новый статус из выпадающего списка
3. Если ставите статус **«Подтверждён»** — обязательно укажите дату и интервал доставки
4. Нажмите **«✅ Сохранить»**

Система автоматически запишет, кто и когда изменил статус — эта информация сохраняется в истории заказа.

### Порядок статусов

```
Новый → В обработке → Подтверждён → Собран → Отправлен → Доставлен → Завершён
                                                           ↘ Отменён (возможно из любого статуса)
```

### Назначение даты доставки

При переводе заказа в статус **«Подтверждён»** нужно указать:
- **Дату** — не раньше завтрашнего дня
- **Временной интервал** — двухчасовое окно (08:00–10:00, 10:00–12:00 и т.д. до 21:00–23:00)

Покупатель увидит эту дату в своём разделе «Заказы».

## 👥 Управление пользователями

Таблица со всеми зарегистрированными пользователями.

### Что видно в таблице

ID, имя, email, телефон (зашифрован), роль (USER или ADMIN), статус аккаунта, дата последнего входа.

### Блокировка и разблокировка

| Кнопка | Когда доступна | Что делает |
|---|---|---|
| **🔓 Разблок.** | Пользователь заблокирован | Снимает блокировку |
| **🔒 Заблок.** | Пользователь активен | Блокирует с указанием причины |

Чтобы заблокировать: нажмите «Заблок.», введите причину в появившемся окне, подтвердите. Заблокированный пользователь не сможет войти в систему.

### Как создать нового администратора

1. Нажмите **«🛡 Добавить администратора»**
2. Заполните форму: имя, email, телефон, пароль
3. Нажмите **«✅ Создать администратора»**

Новый администратор сразу получит доступ к панели управления.

## 📊 Отчёты и аналитика

Раздел для анализа работы магазина.

### Блоки аналитики

Вверху раздела — фильтр по датам (быстрые кнопки 7д/30д/90д, произвольный период, сброс). Ниже — пять блоков:

| Блок | О чём рассказывает |
|---|---|
| 📈 Ключевые показатели (KPI) | Выручка, кол-во заказов, товаров продано, средний чек |
| 👥 Конверсия клиентов | % зарегистрированных, сделавших заказ, повторные покупатели |
| 🏆 Топ-10 товаров | Самые продаваемые товары по выручке (гистограмма) |
| 📂 Продажи по категориям | Распределение выручки по 11 категориям (гистограмма) |
| 📋 Топ-10 по категориям | Плитки с мини-гистограммами для каждой категории |

### Печать отчёта

Нажмите кнопку **«🖨 Печать»** (или `Ctrl+P`) — приложение сформирует HTML-страницу с отчётом и откроет её в браузере. Оттуда можно распечатать или сохранить в PDF.

## 🔧 Частые вопросы

**Как изменить статус заказа?**
Откройте заказ в разделе «Заказы», выберите статус и нажмите «Сохранить».

**Зачем при подтверждении заказа указывать дату доставки?**
Потому что покупатель видит эту дату у себя в «Заказах» и рассчитывает на неё. Без даты заказ подтвердить не получится.

**Можно ли восстановить удалённый товар?**
Нет, удаление окончательное. Если товар временно недоступен, лучше поставьте остаток на складе = 0.

**Пользователь жалуется, что не может войти. Как помочь?**
Откройте «Пользователи», найдите его аккаунт. Если статус «Заблокирован» — нажмите «Разблок.».

**Как добавить ещё одного администратора?**
Раздел «Пользователи» → кнопка «Добавить администратора».

**Как найти нужный заказ?**
В разделе «Заказы» введите номер заказа или имя покупателя в строку поиска.

**Где находится файл базы данных?**
`dist/digitalhub.db` — это один файл, в котором хранятся все данные приложения.
