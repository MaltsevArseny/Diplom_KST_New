## 🚀 Запуск приложения

Чтобы запустить DigitalHub, дважды кликните по файлу **`run.bat`** в корневой папке проекта.

Скрипт сам проверит, что на компьютере установлена Java 21 или выше, соберёт проект и запустит приложение. Никаких дополнительных действий не требуется.

Если проект уже собран, можно запустить быстрее — через файл **`dist/DigitalHub.bat`**.

## 🔐 Вход и регистрация

### Как войти

При запуске откроется экран входа. Введите свой **email** и **пароль**, нажмите **«Войти»**.

Если вы ошибётесь с паролем 5 раз подряд, вход будет временно заблокирован на 5 минут — это сделано для защиты от подбора пароля.

### Как зарегистрироваться

Нажмите ссылку **«Нет аккаунта? Зарегистрироваться»** под формой входа. Откроется форма регистрации.

Что нужно заполнить:

| Поле | Что вводить |
|---|---|
| Имя пользователя | Ваше имя (от 3 до 50 символов) |
| Email | Электронная почта (например, ivanov@mail.ru) |
| Телефон | Номер в формате +7XXXXXXXXXX |
| Пароль | Минимум 8 символов: заглавная буква, строчная, цифра и спецсимвол |
| Подтверждение | Повторите пароль ещё раз |

Поля подсвечиваются при вводе: 🔴 красный — ошибка, 🟢 зелёный — всё правильно. Так вы сразу видите, если что-то заполнено не так.

## 🧭 Навигация по приложению

После входа наверху экрана появляется панель с основными разделами. Можно переключаться между ними кликом или горячими клавишами:

| Раздел | Клавиша | Что там находится |
|---|---|---|
| 🏷 Каталог | `Alt+1` | Все товары магазина |
| 🛒 Корзина | `Alt+2` | Товары, которые вы хотите купить |
| 💜 Избранное | `Alt+3` | Понравившиеся товары |
| 📋 Заказы | `Alt+4` | Ваши оформленные заказы |
| 👤 Профиль | `Alt+5` | Настройки вашего аккаунта |
| ☀ / 🌙 Тема | `Alt+T` | Переключить оформление: тёмное ↔ светлое |
| ❓ Справка | — | Вот это руководство |

### Кнопки управления окном

Приложение использует собственное оформление окна (без стандартной рамки Windows). Кнопки в правом верхнем углу:

| Кнопка | Клавиша | Что делает |
|---|---|---|
| `_` | `Alt+M` | Свернуть в панель задач |
| `□` | — | Развернуть на весь экран или вернуть обратно |
| `×` | `Alt+W` | Закрыть приложение |
| 🚪 Выйти | `Alt+Q` | Выйти из своего аккаунта |

Окно можно перетаскивать, зажав верхнюю панель. Двойной клик по ней — развернуть/восстановить.

### Переключение темы оформления

Рядом с кнопкой «❓ Справка» есть кнопка темы:

- **☀ Светлая** — приложение сейчас в **тёмной** теме. Нажмите, чтобы переключиться на светлую.
- **🌙 Тёмная** — приложение сейчас в **светлой** теме. Нажмите, чтобы переключиться на тёмную.

Сменить тему можно из любого экрана горячей клавишей `Alt+T`. Кнопка темы также доступна в окне входа в систему — выбранную тему приложение запомнит и применит при следующем запуске.

## 🏷 Каталог товаров

### Как искать товары

Вверху каталога расположены инструменты поиска:

- **Строка поиска** — введите название или часть описания
- **Фильтр по категории** — выберите нужную категорию из списка
- **Фильтр по цене** — задайте минимальную и максимальную стоимость
- **Кнопка «Сбросить»** — вернуть все фильтры в начальное состояние

### Как устроен каталог

Все товары разбиты по категориям, отсортированным по алфавиту. У каждой категории свой цвет и значок, чтобы было удобнее ориентироваться:

| Категория | Цвет |
|---|---|
| ⚙ Процессоры | Янтарный |
| 🎮 Видеокарты | Изумрудный |
| 🔌 Материнские платы | Синий |
| 💾 Оперативная память | Фиолетовый |
| 💿 Накопители | Голубой |
| ⚡ Блоки питания | Оранжевый |
| 🖥 Корпуса | Серый |
| ❄ Охлаждение | Циановый |
| 🖱 Периферия | Розовый |
| 🖥 Мониторы | Лаймовый |
| 🌐 Сетевое оборудование | Небесно-голубой |

Категории работают как «аккордеон» — кликните по заголовку, чтобы раскрыть или свернуть список товаров. Сначала показываются первые 15 штук, а для просмотра остальных нажмите **«Показать ещё»**.

### Что можно сделать с товаром

| Действие | Как |
|---|---|
| Добавить в корзину | Нажмите **«В корзину 🛒»** |
| Добавить в избранное | Нажмите на сердечко **♡** |
| Посмотреть подробности | Кликните по названию товара |

Если товар уже в корзине, кнопка покажет **«✓ В корзине»**. Если в избранном — сердечко станет заполненным **❤ Убрать**.

## 🛒 Корзина

В корзине товары сгруппированы по категориям — так проще найти нужный.

### Как менять количество

- Кнопка **+** (зелёная) — увеличить количество. Если товара нет на складе, кнопка станет серой
- Кнопка **−** — уменьшить количество. При количестве = 1 нажатие − предложит удалить товар

### Если случайно удалили товар

Не паникуйте! При удалении появляется кнопка **«Отмена (5с) ↩»**. У вас есть 5 секунд, чтобы вернуть товар на место. После 5 секунд удаление станет окончательным.

### Как оформить заказ

Внизу корзины видна итоговая сумма. Нажмите **«Оформить заказ →»**, чтобы перейти к оформлению.

---

## 💜 Избранное

Здесь хранятся товары, которые вы отметили сердечком. Они тоже сгруппированы по категориям и отсортированы по названию.

На каждой карточке можно:
- **«🛒 В корзину»** — добавить прямо из избранного
- **«× Удалить»** — убрать из избранного (с возможностью отмены в течение 5 секунд)

## 📋 Оформление заказа

После нажатия **«Оформить заказ →»** в корзине откроется форма:

| Поле | Что заполнять |
|---|---|
| Адрес доставки | Город, улица, дом, квартира |
| Контактный телефон | Можно выбрать из ранее использованных или ввести новый |
| Дата доставки | Не раньше завтрашнего дня |
| Интервал доставки | Двухчасовые окна (08:00–10:00, 10:00–12:00 и т.д.) |
| Комментарий | Дополнительные пожелания (необязательно) |

Если вы заказываете не в первый раз, поля заполнятся автоматически из прошлого заказа.

Оплата — **при получении**. Никаких предоплат.

В правой части экрана — сводка заказа: что вы покупаете и на какую сумму.

## 📦 Мои заказы

### Список

Все ваши заказы, сгруппированные по статусу. Можно отфильтровать по конкретному статусу через выпадающий список.

### Какие бывают статусы

| Статус | Что это значит |
|---|---|
| 🔵 Новый | Заказ принят и ожидает обработки |
| 🟡 В обработке | Сотрудник магазина проверяет заказ |
| ✅ Подтверждён | Заказ подтверждён, готовится к отправке |
| 📦 Собран | Заказ упакован и ждёт отправки |
| 🚚 Отправлен | Заказ в пути |
| ✅ Доставлен | Вы получили заказ |
| ✅ Завершён | Заказ закрыт |
| ❌ Отменён | Заказ был отменён |

### Подробности заказа

Кликните по карточке заказа — откроется окно с полной информацией:
- Состав заказа (какие товары, сколько штук, цена)
- Информация о доставке (адрес, телефон, комментарий)
- **Прогресс-трекер** — наглядная шкала, показывающая путь заказа:
  - ✓ зелёный — пройденные этапы (с датой)
  - ● фиолетовый — текущий этап
  - ○ серый — впереди
  - × красный — отменён

### Редактирование данных

Здесь можно изменить имя, email и телефон. После внесения изменений нажмите **«💾 Сохранить данные»**.

2. Придумайте новый (те же требования: 8+ символов, заглавная, строчная, цифра, спецсимвол)
3. Повторите новый пароль
4. Нажмите **«🔒 Изменить пароль»**

## 🔧 Частые вопросы

**Кнопка «+» в корзине серая и не нажимается — почему?**
Скорее всего, на складе закончился этот товар. Количество в корзине не может превышать складской остаток.

**Случайно удалил товар из корзины — можно вернуть?**
Да, если успеете в течение 5 секунд. Нажмите кнопку **«Отмена ↩»**, которая появится на месте удалённого товара.

**Пишет «аккаунт заблокирован» — что делать?**
Это автоматическая блокировка после 5 ошибок ввода пароля. Подождите 5 минут и попробуйте снова. Если блокировка не снимается, обратитесь к администратору магазина.

**Как поменять данные профиля?**
Перейдите в раздел **«Профиль»**, измените нужные поля, нажмите «Сохранить».

**Где посмотреть свои заказы?**
Нажмите **«Заказы»** в верхней панели.

**Как свернуть окно с клавиатуры?**
См. раздел «Горячие клавиши» выше.

**Можно ли войти без регистрации?**
Нет, для работы с приложением необходимо пройти регистрацию.

**Как сделать светлую тему оформления?**
Нажмите кнопку **☀ Светлая** в верхней панели рядом со справкой, либо комбинацию `Alt+T`. Тема запомнится и применится при следующем запуске. Для возврата к тёмной — нажмите **🌙 Тёмная** или `Alt+T` ещё раз.
